Machine Learning

What is machine learning?

What is machine learning? Machine learning is another field in artificial intelligent and computer science, where the machine can mimic the intelligent of a human behaviour through the uses of data and algorithm. In fields such as data science, computer leaning is an important element as prediction and key insight can be uncover with an algorithm of a machine. With this insight application and business have an easier decision-making process. Machine learning process can be broken down into three parts.
1. A Decision process: A perdition or classification is produce using the learning algorithm of a machine. An estimate pattern will be made in the using the data that been stored previously.
2. An Error Function: To know if the prediction is accurate an error function is needed to evaluate the prediction. Example from the past can be used as a comparison to increase the accuracy of the model.
3. A Model Optimization Process: Optimization is where algorithms minimize the differences between the actual data and the prediction data. This helps the algorithms to predict more accurately on new data.

What can be done now?
Machine learning has broadened significantly, as many industry professionals had use it for many different applications. Some of the examples of how machine learning is being used several industries can be seen below:
●Financial Services: To improve their services and satisfaction, the financial services record every customer transaction and engagement to produces data. Using machine learning, they can analyse the pattern of their customer actions, ranging from their spending to saving and investing. This helps them to come up with financial advice and products.
●Retail: Online retailers rely heavily on machine learning to gather and leverage customer data to efficiently build up sales and satisfaction. This is done through gathering information and determining when and where to offers customer discounts, change products or revise pricing. Uber is a perfect example of revise pricing, by using the weather and traffic at the time to determine the pricing in real time. It also helps business to have a better understanding of the consumer demographic, by looking into trend across geographies, over various timespans. This helps them to narrow down and anticipate their customers.
●Healthcare: Medical diagnoses are the main function of machine learning in the medical field. An example would be detecting skin cancer, Stanford has a find a way to diagnosing skin cancer by using algorithm to learn the data that imaging fed into it. Computers are also being trained across the healthcare field to detect any abnormalities or potential issues. Improvement in algorithms can be seen across clinical trials and pharmaceutical. By using multiple sources and combining their data, identify success factors and potential issues simpler.

Will this create, replace, or make redundant any current jobs?
With a rise in machine learning in the job market, there will a rise in unemployment. This is due to computers being able every task from minor and inconvenience to hard complex calculations with little to nonhuman intervention. According to “World Economic Forum” automation displace 75 million jobs worldwide. Thus, gives the employer no reason to hire a team of employees to do the same task as the machine but must pay a hefty amount, while the machine can do it for free. However, this can also create new jobs as well, as often many machines will require a person to operate them. So, employers will look for people who will have the required skills to sustain in the market. While it may seem like machines are taking over the job market, the "World Economic Forum shows that AI can create more jobs than it will destroy, as the report shows that it can create 133 million jobs. Therefore, the long-term effect of machine learning is not as bad as people make it to be.
